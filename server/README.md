# GitBuddy Server

Backend server for GitBuddy application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm run dev
```

## Structure

- `index.js` - Main server entry point
- `routes/` - API routes
- `controllers/` - Request handlers
- `models/` - Data models
- `middleware/` - Custom middleware
- `utils/` - Utility functions
